This repository contains multiple small Node.js/Express topics organized by sections (`section1`, `section2`, `section3`) plus a larger `weekly-project`. The topics cover Socket.IO real-time communication, file streaming, and web application development.

- **Prerequisites**: Node.js (v18+ recommended) and npm installed.
- **Install dependencies**: Run `npm install` inside each topic folder before running it the first time.
- **Run a topic**: From that topic folder, use `node index.js` or `node server.js` or `npm start` (if a start script exists in `package.json`).

### Folders

- **section1**: Socket.IO basics and real-time communication (1.1: Basic Socket.IO setup, 1.2: Chat application, 1.3: Notification system, 1.4: Chat with usernames)
- **section2**: Real-time applications with Socket.IO (2.1: Live sports match simulation, 2.2: Real-time todo synchronization, 2.3: Real-time stock management)
- **section3**: File streaming and real-time updates (3.1: File download with streaming, 3.2: Real-time stock market prices, 3.3: File upload/download with streaming)
- **weekly-project**: E-commerce cart system with Socket.IO, email alerts, and real-time product updates
